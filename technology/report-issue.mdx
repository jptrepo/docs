# Report a Technology Issue

## Submit a Ticket

<Steps>
  <Step title="Classify Your Issue">
    Select the type of issue you're experiencing: - Hardware Problems - Software
    Issues - Network Connectivity - Audio/Visual Equipment - Account Access
  </Step>
  <Step title="Describe the Problem">
    Provide detailed information about: - What you were doing when the issue
    occurred - Any error messages you received - Steps you've already taken
  </Step>
  <Step title="Priority Level">
    Indicate the urgency: - Critical (System Down) - High (Work Blocked) -
    Medium (Limited Functionality) - Low (Minor Issue)
  </Step>
</Steps>
